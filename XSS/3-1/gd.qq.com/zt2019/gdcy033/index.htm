神秘商店：啤酒+故事，像家一样的无人便利店

第33期

导语: 神秘商店，一家连锁规模的无人值守商店。在这里，无论买什么，都像买啤酒似的，总能与故事不期而遇，让你感受到如家般的温暖。

 

-“最后我买了一个泡面味的薯片哈哈哈哈”

-“无人值守，也就没有人说我嗜酒了”

-“一群人把商店洗劫一遍的感觉，真是痛快呀”

-“在这里我遇到了喜欢的女孩，说不定再主动点就有故事啦”

-“买东西会有红包抢，看谁的手气最佳”

-“我想买个男朋友”

-“...”

 

以上这些话，均摘录自神秘商店的留言簿里。一个人的快乐，两个人的邂逅，一群人的狂欢。和商店有过交集的人，仿佛都拥有了一个值得分享的故事。在日益追求商业效率而忽视消费者情感的今天，能让大家收获这么多或惊喜或感动的故事，到底是怎样的一家商店呢？

有责任：希望寻回社会缺失已久的信任感

 

“讲信用的社会，你多久没有经历过？”在神秘商店的宣传片开头，有这样一个问题。对此，神秘商店的创始人杨辉回忆起自己小时候：那会，家家户户离得很近，不仅是物理上的距离近，更是心与心的距离近。每到开饭时间，做菜有多的人家，就会送一些给邻居。人与人之间充盈着温暖和信任。

 

而反观现今社会，信任早已成为社会的稀缺点。出于对“做一份很酷的、有意义的事业”的这份初心的坚守，杨辉希望能够做点什么来寻回人们缺失已久的信任，一点一点地让社会变得更好。于是，便有了“一切源于信任”的神秘商店。

 

有信任：可以促进消费与社交的信用等级

/ 图示：用户留言一 /

 

每一个守信用的人，都能在这里享受到无限便利。

 

重视信任的神秘商店，建立了一个信任互动体系。该体系的价值体现在三个方面，一是：每次使用APP进行支付，都会累积信用积分和生成信用等级（可用于兑换优惠券），优惠券的派发和信用等级的晋升机制会提升用户的消费积极性；二是：信用等级代表着用户的信誉度。因为信任是社交的基础，所以一个人的信誉度越高，相应地也会获得更多的社交机会；三是：用户每完成一次扫码支付动作，都是对自己信任意识的一次强化。久而久之，用户会养成信用洁癖。而这，也是神秘商店没有使用“面部识别”、“指纹识别”等快捷支付方式的原因。

 

因为信任，所以自由。与711、全家等便利店不同，在神秘商店，你可以在冰柜前反复纠结喝什么，而不用介意店员的眼光；与其他无人便利店不同，在神秘商店，出入无需身份验证。就像在家一样，轻松自由。

 

有活力：这里有年轻人最喜欢的商品和活动

/ 图示：用户留言二 /

 

神秘商店是一家很酷的商店。

 

这里聚集了很多有活力有个性的年轻力量，背靠青年公寓，定位年轻消费群体，希望透过品牌传递给年轻人一些精神文化力量。

 

于是，你会发现神秘商店很对年轻人的胃口。线上有APP、社群以及神秘电台来维系用户；线下有神秘音乐之夜：在公寓大堂的舞台上，一个吉他手，一个歌手，漫弹低唱。公寓年轻住户围坐成一圈，享受着神秘商店的啤酒、零食还有音乐。公寓仿佛变成了清吧。

 

而且，这里会定期推出一些定制周边产品、暗黑食品，如神秘月饼、辣椒棒棒糖、火鸡面等。对此，杨辉表示：我们不想通过送手机送现金的方式吸引用户，这么做太粗暴了，和神秘商店的风格调性不相符。那些让人意想不到的、新奇好玩的东西，反而更能得到年轻人青睐。

 

有温度：有人情味儿的用户分享和懂你搭配

/ 图示：用户留言三 /

 

喜欢神秘商店的人，多半是喜欢里面的人情味儿。

 

在这里，最抢手的不是网红零食与潮牌饮料，而是用户分享的手工食品――自制沙拉、手工便当、烘焙蛋糕...神秘商店就像一个公用的大食品柜。这一切，得益于神秘商店独特的用户分享机制：经由门店掌柜的批准后，用户可以在商店里放置自己的分享商品。为了激励用户参与分享，神秘商店没有从中收取费用。在用户的分享互动中，仿佛又看到了旧时邻居间分享饺子的场面。

 

在这里，商店对用户的了解程度甚至远超恋人。通过用户高频的消费行为，神秘商店能够获知用户的消费习惯，并根据这些习惯，做出有温度的销售对策――懂你搭配。比如，某用户特喜欢喝白啤，掌柜会在白啤专柜旁，放上专门搭配白啤的德国香肠；某用户平时喜辣，则会在免费药箱里放一些胃药，并提醒用户“如果胃疼，可以去拿药”。

 

杨辉表示，居住在青年公寓的人们，多半是漂泊他乡的人，而神秘商店能做的，就是通过商品和服务增加用户的归属感。

 

结语: 自15年2月灵感初现，截止至17年11月，神秘商店已经开设了近百家门店，布局以上海、广州、深圳为主。从商业布局和用户评价可以看出，神秘商店已经取得了很不错的成绩。未来，这家看重信任、注重消费体验升级的无人便利店，是否可以继续用“啤酒（消费）”和“故事（社交）”留住用户呢？让我们拭目以待。
